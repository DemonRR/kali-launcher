# kali-launcher

由于个人觉得kali开始菜单使用起来编辑麻烦就想着自己写一个启动器，起初自己手动用Python+PyQt6进行开发，结果效果不是太理想，就让AI基于Electron写了一个。有能力的小伙伴可以二开。

## 安装Electron

```
sudo npm install electron electron-builder --save-dev --verbose
```



## 打包

```
#打包开发版
npm run pack

#打包正式版
npm run dist
```
